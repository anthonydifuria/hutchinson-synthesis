# Examples

8 curated configurations demonstrating specific GIGP regimes and spatial behaviors. Each example includes:

- `config.py` — full parameter configuration to reproduce the result
- `spectrogram.png` — frequency distribution over time
- `audio_binaural.wav` — binaural render for headphone listening

| # | Name | Key Feature |
|---|------|-------------|
| 01 | dpp_3niches | DPP repulsion across 3 frequency niches |
| 02 | permanental_cluster | Permanental attraction, timbral fusion |
| 03 | hawkes_temporal | Hawkes ordered dependence in time |
| 04 | lgcp_heterogeneous | LGCP heterogeneous intensity field |
| 05 | alpha_stable_sparse | Alpha-Stable heavy tails, sparse events |
| 06 | beta_ensemble | Intermediate β regime |
| 07 | ren_spatial | REN non-convex spatial boundary |
| 08 | rings_elevation | Elevation rings spatial constraint |
